On Call Bot

I built a Slack app for assigning engineers to be "on call" for monolith release nights. It also automatically creates Slack reminders for the engineers on call, and automatically assigns engineers to new releases as they are created.

This was my first big backend solo project, and it was super fun to make. I had a lot of trouble figuring out timezone issues when creating reminders after I'd deployed the Node server to Now, but I think everyone struggles with times in JavaScript at some point... I ended up reaching for moment.js just to simplify my life, even though it's a relatively heavy dependency.
